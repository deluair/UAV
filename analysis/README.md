# China UAV Market Analysis

Run the market analysis and projections:

```bash
python analysis/china_market_analysis.py
```

This will generate:
- 2026 market projections
- Scenario analysis (optimistic, baseline, pessimistic)
- Forecast data CSV file
- Visualization charts

## Output Files

- `china_uav_forecast_2024_2026.csv` - Detailed forecast data
- `china_uav_projections.png` - Visualization charts

## Key Insights

- China maintains 70% market share through 2026
- Global market reaches $38.2 billion by 2026
- China's total market value: $18-20 billion
- Manufacturing capacity: 12+ million units annually
